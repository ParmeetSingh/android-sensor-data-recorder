Sensor Data Recorder

Android application to record various sensor data in a write-ahead only file with high frequency writes

View on GitHub Download .zip Download .tar.gz

Introduction

This application was created during our Thesis project for our bachelor degree. The idea was to create an Android application which could record the raw data given by different sensors. This data can be further analysed for various purposes. In our case, it was to detect potholes on roads.

Sensors

We've used support for recording data from following sensors. All these sensors can be listened to by simply changing a few lines in the code.

  • GPS
  • Accelerometer
  • Magnetometer
  • Gyroscope

Features

This app contains a lot of useful features which are very essential for the developers. Auto start/stop recording, High Speed data recording in file, and upload to server to name a few.

Auto Stop/Start Recording

The greatest feature that we designed was to start and stop the recording based on the movement. Movement was detected by pinging GPS after some amount of time which was configurable through settings.

Server File-Upload

You can upload your recorded data to server provided there is a script handling the file upload. At the time of designing this application, we used a PHP server. The IP-Address of the server is also configurable through settings.

High Speed File Writing

This application records data in a file on the Android filesystem. File is write-ahead only. It was designed in a such a way that it shouldn't consume too much memory and still be able to append data at a very fast rate. (500 writes per second on average)
